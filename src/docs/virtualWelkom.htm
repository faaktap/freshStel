  3. Select Wi-Fi.
  4. Make sure the Wi-Fi is switched on.
  5. Once the Wi-Fi is on, select the WCGSCHOOLS network.
  6. Add the Identity and password fields. Note: Leave Anonymous identity blank.
  7. Scroll down and make sure show advanced options is ticked.
  8. Select the proxy type and add the proxy hostname and proxy port.
